## Part 6*. Уведомления

### Настроим уведомления об успешном/неуспешном выполнении пайплайна через бота с именем «[nickname] DO6 CI/CD» в *Telegram*

1. Найдём в *Telegram* бота *BotFather* и отправим ему сообщение `/newbot`. Зададим название и id нашего бота, в ответ получим токен для дальнейшей работы.

  ![](images/part6/1.png)

2. Найдём нашего бота и что-нибудь ему отправим.

  ![](images/part6/2.png)

3. Перейдём по ссылке *https://api.telegram.org/botTOKEN/getUpdates*, где вместо *TOKEN* будет, собсна, выданный нам в первом шаге токен. Там найдём id чата нашего новорождённого бота. В нашем случае это **871565755**.

  ![](images/part6/3.png)

4. Пишем скрипт для уведомлений.

![](images/part6/4.png)

5. После каждой стадии вызываем этот скрипт.

![](images/part6/5.png)

6. Пушим все изменения, и бот начинает нам писать всякое:

![](images/part6/6.png)
